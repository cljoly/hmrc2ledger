# Hmrc2ledger

***Disclaimer*: This is not financial advice. It may be outdated or wrong, do your own research.**

Convert [CSV][csv] to [ledger][] price db, to compute [foreign](sterling) transaction CGT (and [section 104][s104] holdings).
